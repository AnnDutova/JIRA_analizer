Jira analyzer

 

Данное приложение позволяет осуществить аналитику по проектам из открытых источников Jira.


Добавить проекты для аналитики вы можете на странице "Проекты".


Посмотреть статистику по скачанным проектам и выполнить аналитические задачи для выбранных проектов вы можете через страницу "Меню". Результат выполнения аналитических задач будет представлен в виде графиков. Для выполнения доступны 6 различных аналитических задач:

  1. Гистограмма, отражающая время, которое задачи провели в открытом состоянии.
  2. Диаграммы, которые показывают распределение времени по состоянием задач.
  3. График активности по задачам (показывает количество заведенных и закрытых задач в день).
  4. График сложности задач (отражает залогированное время, которое потратил участник на ее выполнение).
  5. График, отражающий приоритетность всех задач.
  6. График, отражающий приоритетность закрытых задач.

 

Провести сравнение по проектам можно перейдя на страницу "Сравнение". Обратите внимание на то, что в результате сравнения проектов будут построены только те сводные диаграммы, аналитика, для которых вызывалась у всех проектов, которые вы хотите сравнить.
